AngularJS

AngularJS is a toolset for building the framework most suited to your application development. It is highly versatile and we can build a common set of components for re-use in future projects.

Firebase

Firebase is a real-time back-end for front-end developers like us. We can focus on the UI and the client side application more.

Ionic

Ionic Framework is based on Cordova/Phonegap and the AngularJS framework. This means that it is possible to build hybrid apps for all mobile devices based on one code-base. Totally ingenious and all rather exciting!
